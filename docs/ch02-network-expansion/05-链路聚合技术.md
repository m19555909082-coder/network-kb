### 📄 Slide 21：办公网络优化之----链路聚合技术Link aggregation techniques for office network optimization

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

链路聚合是将两个或更多数据信道结合成一个单个的信道，该信道以一个单个的更高带宽的逻辑链路出现。链路聚合一般用来连接一个或多个带宽需求大的设备，其实质是将两台设备间的数条物理链路“组合”成逻辑上的一条数据通路，称为一条聚合链路AP1。该链路在逻辑上是一个整体。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

办公网络优化之----链路聚合技术Link aggregation techniques for office network optimization
Link aggregation is the combination of two or more data channels into a single channel that emerges as a single higher bandwidth logical link. Link aggregation is generally used to connect one or more devices with large bandwidth requirements. In essence, it "combines" several physical links between two devices into a logical data path, called an aggregation link AP1. The link is logically a whole.

</div>

### 📄 Slide 22：配置链路聚合（端口聚合）的基本命令如下:

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

配置链路聚合（端口聚合）的基本命令如下:
Switch(config-if-range)# channel-group  1 mode on（将交换机的物理端口加入到一个名为 channel-group 1 的端口聚合组中，并将其设置为“on”模式）
Switch#show etherchannel （显示 EtherChannel 配置和状态的命令，查看 EtherChannel 的详细信息，包括成员端口、通道状态、负载均衡策略等）

Switch(config-if-range)# exit
Switch(config)#int  port-channel  1
