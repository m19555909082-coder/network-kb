### 📄 Slide 38：路由器实现VLAN间路由The router implements inter-VLAN routing

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

利用路由器的路由功能，实现不同子接口数据的转发
缺点是：部署不灵活，形成网络瓶颈。
在路由器上为每个VLAN划分一个子接口
每个子接口配置IP地址，作为相应VLAN的网关

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

路由器实现VLAN间路由The router implements inter-VLAN routing
The routing function of the router is used to realize the forwarding of data from different sub-interfaces
The disadvantages are: inflexible deployment, forming a network bottleneck.
A subinterface is divided at the router for each VLAN
Each subinterface is configured with an IP address that acts as a gateway to the corresponding VLAN

</div>
**💻 配置命令：**

```cisco
路由器实现VLAN间路由The router implements inter-VLAN routing
The disadvantages are: inflexible deployment, forming a network bottleneck.
A subinterface is divided at the router for each VLAN
Each subinterface is configured with an IP address that acts as a gateway to the corresponding VLAN
```

### 📄 Slide 39：单臂路由配置

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

单臂路由配置
步骤1：创建以太网子接口
步骤2：为子接口封装802.1q协议，并指定接口所属的VLAN
步骤3：为子接口配置IP地址
步骤4：启用子接口

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Router(config)#interface interface.sub-port
步骤2：为子接口封装802.1q协议，并指定接口所属的VLAN
Router(config-subif)#encapsulation dot1q vlan-id
Router(config-subif)#ip address ip-address mask-address
Router(config-subif)#no shutdown

</div>
**💻 配置命令：**

```cisco
Router(config)#interface interface.sub-port
Router(config-subif)#encapsulation dot1q vlan-id
Router(config-subif)#ip address ip-address mask-address
Router(config-subif)#no shutdown
```

### 📄 Slide 40：单臂路由配置示例Example single-arm routing configuration

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

单臂路由配置示例Example single-arm routing configuration
Router(config)#interface fastEthernet 0/0
Router(config-if)#no shutdown（思科路由器要有）

Router(config)#interface fastEthernet 0/0.1
Router(config-subif)#encapsulation dot1q 10
Router(config-subif)#ip address 192.168.1.1 255.255.255.0
Router(config-subif)#no shutdown
Router(config-subif)#exit

Router(config)#interface fastEthernet 0/0.2
Router(config-subif)#encapsulation dot1q 20
Router(config-subif)#ip address 192.168.2.1 255.255.255.0
Router(config-subif)#no shutdown
Router(config-subif)#end
