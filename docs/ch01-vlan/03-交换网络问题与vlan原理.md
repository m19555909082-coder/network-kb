### 📄 Slide 11：虚拟局域网

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

虚拟局域网
VLAN为解决以太网广播问题和安全问题而提出解决方案，它在以太网全网广播基础上，把用户划分到更小工作组中，每个工作组就相当于一个隔离的局域网。这些隔离的局域网的好处是可以限制广播范围，形成虚拟工作组。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

VLAN
VLAN provides a solution to solve the broadcast problem and security problem of Ethernet. Based on the broadcast of Ethernet, VLAN divides users into smaller workgroups, and each workgroup is equivalent to an isolated local area network. The benefit of these isolated Lans is that the broadcast range can be limited and virtual workgroups can be formed.

</div>
