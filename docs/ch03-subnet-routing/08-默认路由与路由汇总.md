### 📄 Slide 48：接口IP地址配置的基本原则

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

接口IP地址配置的基本原则
路由器的物理网络端口需要有一个IP地址
相邻路由器的相邻端口IP地址在同一网段
同一路由器不同端口在不同网段上

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Basic principles for interface IP address configuration
The physical network port of the router needs to have an IP address
Adjacent port IP addresses of adjacent routers are in the same network segment
The same router has different ports on different network segments

</div>
**💻 配置命令：**

```cisco
Basic principles for interface IP address configuration
The physical network port of the router needs to have an IP address
Adjacent port IP addresses of adjacent routers are in the same network segment
The same router has different ports on different network segments
```
