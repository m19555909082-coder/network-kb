### 📄 Slide 20：可以用来选择采用哪个IP路由协议

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

可以用来选择采用哪个IP路由协议
值越低，学到的路由越可信
静态配置路由优先于动态协议学到的路由
采用复杂量度的路由协议优先于简单量度的路由协议。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Can be used to choose which IP routing protocol to use
The lower the value, the more credible the learned route
Static configured routes take precedence over routes learned by dynamic protocols
Routing protocols with complex metrics are preferred to those with simple metrics.
管理距离(可信度)Managing distance (credibility)

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

路由器实际上也是一台特殊的通信计算机，由硬件系统和软件系统构成。硬件结构包括：内部的处理器、存储器和各种不同类型接口。操作系统控制软件是控制路由器硬件工作的核心。
路由器设备组成

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Router is actually a special communication computer, which is composed of hardware system and software system. Hardware structure includes: internal processor, memory and various types of interfaces. Operating system control software is the core of controlling router hardware.
Composition of router device
