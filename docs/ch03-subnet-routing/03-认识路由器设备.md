<div class="bilingual-cn" markdown>

**🇨🇳 中文**

连通不同网络的设备—路由器
路由器是一种连接多个不同网络或子网段的网络互连设备，如图所示。路由器中的“路由”是指在相互连接的多个网络中，信息从源网络移动到目标网络的活动。一般来说数据包在路由过程中，至少经过一个以上的中间节点设备。路由器为经过其上的每个数据包寻找一条最佳传输路径，以保证该数据有效、快速地传送到目的计算机。
认识路由器设备

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

A device that connects different networks - a router
A router is a network interconnection device that connects multiple different networks or subnetwork segments, as shown in Figure . "Routing" in routers refers to the activity of moving information from a source network to a destination network in multiple interconnected networks. Generally speaking, data packets pass through at least one intermediate node device during the routing process. Routers find an optimal path for each packet that passes through them, so that the data can be delivered to the destination computer efficiently and quickly.

<div class="bilingual-en" markdown>

**🇬🇧 English**

The router saves the address information Table of various transmission paths, commonly known as Routing Table, for packet routing. The routing table stores the identification information of each subnet: route identification, route method, destination network, forwarding router address and the number of passing routers.

### 📄 Slide 16：路由器还可以使用不同协议和体系结构的网络中，当数据信息从一种网络传输到另外一种类型网络时，路由器接收来自不同类型网络的数据信息，通过分析数据包中携带的信息，阅读...

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

路由器还可以使用不同协议和体系结构的网络中，当数据信息从一种网络传输到另外一种类型网络时，路由器接收来自不同类型网络的数据信息，通过分析数据包中携带的信息，阅读、翻译，以使它们能够接受到，或者相互“读”懂对方的数据，从而构成所有网络的互联互通。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Routers can also use networks with different protocols and architectures. When data is transmitted from one type of network to another type of network, routers receive data from different types of networks, read and translate the information carried in the data packets so that they can receive, or "read" each other's data. Thus constitutes the interconnection of all networks.

</div>

### 📄 Slide 17：在路由器的内部都有一个路由表，这个路由表中包含有该路由器知道的目的网络地址以及通过此路由器到达这些网络的最佳路径，如某个接口或下一跳的地址，正是由于路由表的存在...

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

在路由器的内部都有一个路由表，这个路由表中包含有该路由器知道的目的网络地址以及通过此路由器到达这些网络的最佳路径，如某个接口或下一跳的地址，正是由于路由表的存在，路由器可以依据它进行转发。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

In the interior of a router, there is a routing table that contains the destination network addresses known to the router and the best path through the router to reach these networks, such as the address of a certain interface or next hop. Because of the existence of the routing table, the router can forward according to it.

</div>
