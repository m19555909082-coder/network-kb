### 📄 Slide 3：项目1：使用路由器实现子网连通Project 1: Implement subnet connectivity using router

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

学校网络中心为解决校内广播和安全问题，对学校原有的网络进行重新规划，通过划子网方式来隔离网络中广播。校园网改造的过程中，决定使用路由设备连接多个分散的子网络，通过直连路由技术实现分散的不同子网系统之间互连互通。
【任务目标】：task object
直连路由技术。

</div>

**🇬🇧 English**

项目1：使用路由器实现子网连通Project 1: Implement subnet connectivity using router
【任务描述】task description
In order to solve the problem of broadcasting and safety in the school, the school network center replanned the original network and isolated the broadcasting in the network through the way of sub-net. In the process of campus network reconstruction, it is decided to use routing device  to connect multiple scattered sub-networks, and realize the interconnection between different scattered sub-network systems through directly connected  routing technology.
