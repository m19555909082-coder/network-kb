### 📄 Slide 8：什么是路由：What is a route

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

所谓路由就是指通过相互连接的网络，把数据从源地点转发到目标地点的过程。一般来说，数据在网络中路由的过程，数据至少会经过一个或多个中间节点。路由技术发生在OSI模型的第三层（网络层），路由包含两个基本的动作：（首先存储数据）确定最佳路径和通过网络传输信息，后者也称为数据转发。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

什么是路由：What is a route
Routing is the process of forwarding data from a source location to a destination location through an interconnected network. In general, the process of data routing in the network, the data will pass through at least one or more intermediate nodes. Routing technology occurs in the third layer of the OSI model (network layer), routing contains two basic actions: (first storing data) determining the best path and transmitting information through the network, the latter is also called data forwarding.

</div>

### 📄 Slide 11：当IP子网中的一台主机，需要发送一个IP数据包给同一IP子网的另一台主机时，如果接受数据包的主机在同一个子网中，通过广播（都是广播吗）对方就能收到。而要一个IP...

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

当IP子网中的一台主机，需要发送一个IP数据包给同一IP子网的另一台主机时，如果接受数据包的主机在同一个子网中，通过广播（都是广播吗）对方就能收到。而要一个IP分组发送给不同IP子网上的主机时，发送主机通过目标IP地址与子网掩码运算，判断通信双方不在同一子网，因此数据分组将被转发至发送主机的缺省网关，也就是它要选择一个能到达目的子网上的三层路由设备。
网络层路由工作原理

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

When a host in the IP subnet needs to send an IP packet to another host in the same IP subnet, if the receiving host is in the same subnet, the other host can receive the packet by broadcasting. When an IP packet is sent to a host on a different IP subnet, the sending host determines that the communication parties are not in the same subnet by calculating the target IP address and subnet mask, so the data packet will be forwarded to the default gateway of the sending host, that is, it has to select a layer 3 routing device that can reach the destination subnet.
How network layer routing works

</div>
