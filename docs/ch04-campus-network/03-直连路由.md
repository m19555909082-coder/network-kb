### 📄 Slide 15：园区网静态路由Static routing in campus network

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

静态路由是在路由器中设置的固定路由表。除非网络管理员干预，否则静态路由不会发生变化。由于静态路由不能对网络的改变作出反映，一般用于网络规模不大、拓扑结构固定的网络中。
静态路由的优点是简单、高效、可靠。在所有的路由中，静态路由优先级最高（管理距离为0或1）。当动态路由与静态路由发生冲突时，以静态路由为准。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

园区网静态路由Static routing in campus network
静态路由  Static routing
Static routing is a fixed routing table set in the router. Static routing does not change unless the network administrator intervenes. Because static routing can not reflect the changes of the network, it is generally used in the network with small scale and fixed topology.
The advantages of static routing are simplicity, efficiency, and reliability. Of all the routes, static routes have the highest priority (administrative distance 0 or 1). When a dynamic route conflicts with a static route, the static route takes precedence.

</div>

### 📄 Slide 19：静态路由配置命令

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

静态路由配置命令
配置静态路由用命令ip route
router(config)#ip route [网络编号] [子网掩码] [转发路由器的IP地址/本地接口]
静态路由描述转发路径的方式有两种
指向本地接口（即从本地某接口发出）
指向下一跳路由器直连接口的IP地址（即将数据包交给X.X.X.X）

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Static route configuration commands
配置静态路由用命令ip route
Commands to configure static routes: ip route
router(config)#ip route [网络编号] [子网掩码] [转发路由器的IP地址/本地接口]
router(config)#ip route [Network number] [subnet mask] [IP address of forwarding router/local interface]
example：ip route 192.168.10.0 255.255.255.0  serial 0
There are two ways in which static routes describe forwarding paths
Points to a local interface (i.e., from a local interface)
The IP address that points to the direct connection port of the next router (that is, deliver the packet to X.X.X.X)

### 📄 Slide 20：静态路由Static Routing

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

静态路由的一般配置步骤
1.为路由器每个接口配置IP地址
2.确定本路由器有哪些直连网段的路由信息
3.确定网络中有哪些属于本路由器的非直连网段
4.添加本路由器的非直连网段相关的路由信息

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

静态路由Static Routing
General configuration steps for static routing
1.Configure IP address for each interface of router
2. Determine which direct network segments the router has routing information
3. Identify the non-direct segments of the network that belong to the router
4. Add routing information related to non-direct network segments of the router

</div>
**💻 配置命令：**

```cisco
1.Configure IP address for each interface of router
2. Determine which direct network segments the router has routing information
3. Identify the non-direct segments of the network that belong to the router
4. Add routing information related to non-direct network segments of the router
```
