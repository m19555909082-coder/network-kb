### 📄 Slide 26：园区网RIP动态路由协议RIP dynamic routing protocol for campus network

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

动态路由协议（Routing Protocol）用于路由器动态寻找网络最佳路径，保证所有路由器拥有相同的路由表，一般路由协议决定数据包在网络上的行走路径，这类协议的例子有OSPF，RIP等路由协议。路由选择协议消息在路由器之间传送，允许路由器与其他路由器通信，生成、更新和维护路由选择表。

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

园区网RIP动态路由协议RIP dynamic routing protocol for campus network
Dynamic Routing Protocol (Routing Protocol) is used for routers to dynamically find the best path in the network, to ensure that all routers have the same routing table, the general routing protocol determines the packet on the network path, such as OSPF, RIP routing protocol and other examples. Routing protocol messages are passed between routers, allowing routers to communicate with other routers, generate, update, and maintain routing tables.

</div>
