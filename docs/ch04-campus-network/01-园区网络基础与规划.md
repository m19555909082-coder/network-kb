### 📄 Slide 5：园区网核心层的功能主要是实现园区骨干网络之间的优化传输，园区网骨干设计任务的重点通常是冗余能力、可靠性和高速的传输。网络的控制功能最好尽量少在园区网骨干层上实施...

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

园区网核心层的功能主要是实现园区骨干网络之间的优化传输，园区网骨干设计任务的重点通常是冗余能力、可靠性和高速的传输。网络的控制功能最好尽量少在园区网骨干层上实施。核心层一直被认为是所有流量的最终承受者和汇聚者，所以对园区网核心层的设计以及网络设备的选型上要求十分严格。一般园区网核心层设备将占投资的主要部分。
园区网络规划和设计

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

The main function of the core layer of the campus network is to realize the optimal transmission between the backbone networks of the campus network. The focus of the backbone design tasks is usually redundancy, reliability and high-speed transmission. The control functions of the network should be implemented as little as possible on the backbone layer of the campus network. The core layer has been considered as the ultimate recipient and aggregator of all traffic, so the design of the core layer and the selection of network equipment are very strict. General campus network core layer equipment will account for the main part of the investment.
Campus network planning and design

</div>

<div class="bilingual-cn" markdown>

**🇨🇳 中文**

园区网汇聚层提供园区网络基于统一策略互连，汇聚层是核心层和访问层分界点，定义了网络的边界，对数据包进行复杂运算。在园区网络环境中，汇聚层主要提供地址聚集、部门和工作组接入 、广播域/多目传输域定义 、VLAN路由 、安全控制等功能。
园区网接入层主要功能是为最终用户提供对园区网络访问途径。在园区网络环境中，接入层主要提供资源共享、数据交换、MAC地址交换等功能。
园区网络规划和设计

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

The aggregation layer of campus network provides the interconnection of campus network based on uniform policy. The aggregation layer is the demarcation point between the core layer and the access layer, defines the boundary of the network, and performs complex operations on data packets. In the campus network environment, the aggregation layer mainly provides address aggregation, department and work group access, broadcast domain/multicast domain definition, VLAN routing, security control and other functions.
The main function of campus network access layer is to provide access to campus network for end users. In the campus network environment, the access layer mainly provides resources sharing, data exchange, MAC address exchange and other functions.
Campus network planning and design

</div>

**🇨🇳 中文**

有时为了方便网络管理，需要将网络划分为若干个网段。为此必须打破传统8位界限，从结点地址空间中“抢来”几位作为网络地址。
具体说来，建立子网掩码需要以下两步：
确定运行IP的网段数
确定子网掩码
园区网络IP地址规划

</div>

<div class="bilingual-en" markdown>

**🇬🇧 English**

Sometimes for the convenience of network management, it is necessary to divide the network into several network segments. Therefore, it is necessary to break the traditional 8-bit boundary and grab several bits from the node address space as network addresses.
In particular, creating a subnet mask requires the following two steps:
Determine the number of network segments on which the IP is running
Determine the subnet mask
Campus network IP address planning
