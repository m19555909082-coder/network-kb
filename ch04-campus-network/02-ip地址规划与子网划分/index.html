    📄 Slide 8：路由概念：略 Routing concept: Omitted¶

    🇨🇳 中文

    路由发生在网络层,包含两个基本动作：确定最佳路径和通过网络转发数据。 网络中的数据包通过路由器转发到目的网络，必须依据于路由表，路由表中包含有目的网络地址以及到达这些网络的最佳路径，如某个接口或下一跳地址。正是由于路由表的存在，路由器可以依据它进行数据转发。

    🇬🇧 English

    路由概念：略 Routing concept: Omitted Routing occurs at the network layer and consists of two basic actions: determining the best path and forwarding data through the network. Packets in the network are forwarded to the destination network by the router according to the routing table, which contains the destination network address and the best path to reach these networks, such as a certain interface or next hop address. Because of the existence of routing table, routers can forward data according to it. 典型的路由表产生方式有三种：Routing tables are typically generated in three ways: 直连路由 Directly connected routing 静态路由 Static routing 动态路由 Dynamic routing 园区网路由Campus network routing

    📄 Slide 11：园区网直连路由 Campus network directly connected routing¶

    🇨🇳 中文

    直连路由是在配置完路由器网络接口的IP地址后自动生成的，因此如果没有对这些接口进行特殊的限制，这些接口所直连的网络之间就可以直接通信。 一般把在路由器接口所连接的子网，直接配置地址的路由方式称为直连路由，直连路由基本功能就是实现邻居的互通。

    🇬🇧 English

    园区网直连路由 Campus network directly connected routing 直连路由 directly connected routing Directly connected routing is automatically generated after the IP addresses of the router's network interfaces are configured, so that the networks to which these interfaces are directly connected can communicate with each other if no special restrictions are placed on these interfaces. In general, the routing method of directly configuring the address in the subnet connected by the router interface is called directly connected routing. The basic function of directly connected routing is to realize the intercommunication of neighbors.

    💻 配置命令：

    园区网直连路由Campus network directly connected routing
Directly connected routing is automatically generated after the IP addresses of the router's network interfaces are configured, so that the networks to which these interfaces are directly connected can communicate with each other if no special restrictions are placed on these interfaces.
In general, the routing method of directly configuring the address in the subnet connected by the router interface is called directly connected routing. The basic function of directly connected routing is to realize the intercommunication of neighbors.
